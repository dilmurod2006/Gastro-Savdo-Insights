# Notebooks Directory

Bu papkada Jupyter Notebook fayllari saqlanadi.

## Maqsad

Jupyter Notebook'lar quyidagi vazifalar uchun ishlatiladi:
- Ma'lumotlarni tadqiq qilish (Exploratory Data Analysis - EDA)
- Ma'lumotlarni tozalash va qayta ishlash
- Modellarni sinash va prototiping
- Vizualizatsiyalarni yaratish
- Tahlil natijalari va hisobotlar

## Foydalanish

Jupyter Notebook ishga tushirish uchun:

```bash
# Jupyter Lab ni o'rnatish
pip install jupyterlab

# Jupyter Lab ni ishga tushirish
jupyter lab
```

## Notebook Nomlash Konvensiyasi

Notebook fayllarini quyidagicha nomlash tavsiya etiladi:
- `01_data_exploration.ipynb` - Ma'lumotlarni o'rganish
- `02_data_cleaning.ipynb` - Ma'lumotlarni tozalash
- `03_data_analysis.ipynb` - Ma'lumotlar tahlili
- `04_visualization.ipynb` - Vizualizatsiya
- `05_modeling.ipynb` - Modellashtirish

## Eslatma

⚠️ Notebook fayllaridagi katta ma'lumotlar va chiqishlarni commit qilmaslik uchun:
1. Notebook'ni `Clear All Outputs` qiling
2. Yoki `.gitignore` faylida kerakli qoidalarni qo'shing
